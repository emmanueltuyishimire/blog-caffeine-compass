# **App Name**: Caffeine Chronicle

## Core Features:

- Blog Posts: Display blog posts related to coffee and caffeine.
- Product Reviews: Show user reviews of various coffee and caffeine products.
- Caffeine Calculator: Tool that allow user to estimate safe caffeine intakes based on their personal data and preferences
- Route Handling: Ensure the app works correctly under the /blog directory of the calculation.site domain (https://www.calculation.site/blog).
- AI Post Summarizer: Use AI to generate summaries for long blog posts.

## Style Guidelines:

- Primary color: Warm brown (#A67B5B) to evoke a sense of coffee and comfort.
- Background color: Light beige (#F5F5DC), a desaturated tint of the primary.
- Accent color: Golden yellow (#FFD700) for highlights and call-to-action buttons.
- Body text: 'Literata' (serif), creating a vintage, slightly formal feel appropriate for long-form editorial content.
- Headline font: 'Space Grotesk' (sans-serif) providing computerized, techy, scientific feel; suitable for headlines, that contrast with Literata.
- Use coffee-related icons, such as coffee beans, cups, and pots.
- Clean and readable layout with ample whitespace.
- Subtle transitions and animations to enhance user experience.
